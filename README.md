# QuickChat Application

A Java-based chat application with user registration, authentication, and messaging system.

## Features
- User registration with validation
- Login authentication  
- Message sending with JSON storage
- Cell phone number validation
- Password complexity checking
- Data management and reporting (Part 3)
- Unit testing system

## Part 3 Features Added
- **Data Management**: Arrays for sent, stored, and disregarded messages
- **Message Search**: Search by ID, recipient, and message content
- **Reporting**: Full message reports with statistics
- **Message Operations**: Delete messages by hash, find longest messages
- **Unit Tests**: Comprehensive test suite for all features

## Files Structure
- `Main.java` - Application entry point
- `Registration.java` - User registration logic
- `Login.java` - User authentication
- `ChatApplication.java` - Chat system management
- `Message.java` - Message handling and validation
- `SimpleJSONHandler.java` - JSON file storage
- `DataManager.java` - Data management and arrays (Part 3)
- `TestManager.java` - Unit tests (Part 3)
