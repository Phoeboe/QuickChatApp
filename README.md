# QuickChat Application

A Java-based chat application with user registration, authentication, and messaging system.

## Features
- User registration with validation
- Login authentication  
- Message sending with JSON storage
- Cell phone number validation
- Password complexity checking

## How to Run
1. Compile all Java files: `javac *.java`
2. Run the application: `java Main`

## Files
- `Main.java` - Application entry point
- `Registration.java` - User registration
- `Login.java` - User authentication  
- `ChatApplication.java` - Chat system
- `Message.java` - Message handling
- `SimpleJSONHandler.java` - JSON storage
